Installation en 1 clic · aucun store requis

Téléchargez SnapFiche

Choisissez votre appareil. SnapFiche s'installe comme une vraie application, fonctionne hors ligne pour la navigation et reste toujours à jour.

Android

Chrome / Samsung Internet
  1. Touchez « Installer » ci-dessous (ou le menu ⋮ → « Installer l'application »).
  2. Confirmez « Ajouter à l'écran d'accueil ».
  3. L'icône SnapFiche apparaît comme une app.

iPhone / iPad

Safari
  1. Ouvrez snapfiche.com dans Safari.
  2. Touchez le bouton Partager (carré avec une flèche).
  3. Choisissez « Sur l'écran d'accueil » puis « Ajouter ».
Ouvrir dans Safari

Windows

Edge / Chrome
  1. Cliquez « Installer » ci-dessous (ou l'icône d'installation dans la barre d'adresse).
  2. Ou menu ⋮ → « Applications » → « Installer SnapFiche ».
  3. L'app s'ouvre dans sa propre fenêtre, épinglable.

Application web (PWA)

Tout navigateur moderne
  1. Pas d'installation ? Utilisez SnapFiche directement dans le navigateur.
  2. Cliquez « Installer » pour l'ajouter à votre appareil.
  3. Fonctionne sur Mac, Linux, Chromebook…
Pourquoi une application web ? SnapFiche est une PWA : pas besoin d'App Store ni de Play Store, installation en 1 clic, mises à jour automatiques, et la même expérience sur téléphone et ordinateur.
